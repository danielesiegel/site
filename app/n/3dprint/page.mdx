# Print Farm Tracker

<AnimatedName />

### Robotic Print Farm Tracker

Developed a prototype for a robotic system to monitor progress in a 3D-printing farm. The device moves along a modular rail system and provides real-time updates through a web interface. Designed and assembled the system in SOLIDWORKS with over 50 unique components.

---

### Key Features

- Modular rail system for navigating the print farm.
- Real-time web-based tracking for print updates.
- Mechanical design created in SOLIDWORKS and assembled to specifications.
