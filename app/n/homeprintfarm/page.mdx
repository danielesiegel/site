# Home Print Farm

<AnimatedName />

### Home Print Farm

In early 2024, I acquired a few 3D printers and a laser cutter for free. I decided to create a custom table and modify my workshop shed to accommodate these machines. The table was designed in SOLIDWORKS, optimized for sturdiness, and adjusted to my height for comfortable standing work. It also includes storage and space for the printers.

---

### Key Features

- **Klipper Integration**: System optimized with Klipper and OrangePi running Debian, accessible via a web server.
- **25-Point Calibration**: Ensures stable prints, compatible with Klipper.
- **Enclosed Printing Environments**: Humidity-controlled enclosures for materials like PETG and ABS.
- **Custom Lighting**: Installed six 1100-lumen fixtures for a bright workspace.
- **Soldering Station**: Dedicated area for electronic work.

---

### Printers and Laser Cutter

- Sovol SV07 Plus
- 2x Ender 3 (Modified with Klipper and server fan cooling)
- Elegoo Mars 2 Pro SLA Printer
- Unnamed Chinese Laser Cutter
